# 介绍
本项目是个人博客的前端项目，服务端项目[请点击](https://github.com/chaikd/blog-serve.git)
# 在线示例
  [Preview](blog.chaikd.com)
# 项目启动
	# clone the project
  git clone https://github.com/chaikd/blog.git

  # enter the project directory
  cd blog

  # install dependency
  npm install

  # develop
  npm run serve
# 项目打包
  npm run build
# 功能
